# XLogin

A simple React login interface with username and password fields.

## Features

- Uses a form element with proper labels for Username and Password.
- Validates that both fields are filled.
- Shows **"Welcome, user!"** when username is `user` and password is `password`.
- Shows **"Invalid username or password"** for any incorrect input.

## How to Run

1. Install dependencies: npm install
2. Start the application: npm start (The app will run on `http://localhost:3000`.)
